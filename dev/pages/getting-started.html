Getting started

Install

using Pkg
Pkg.add(url = "https://github.com/joshday/GoogleMapsAPI.jl")

Authenticate

Every function reads ENV["GOOGLE_MAPS_API_KEY"] by default. Export it in your shell:

export GOOGLE_MAPS_API_KEY="your-key-here"

Or pass key = "..." (or a preconfigured client = GoogleMapsClient(...)) to any call.

First call

using GoogleMapsAPI

r = geocode("1600 Amphitheatre Pkwy, Mountain View, CA")
r.results[1].geometry.location   # (lat = ..., lng = ...)

route = compute_routes(
    (35.9132, -79.0822),            # Carrboro, NC
    (38.9805, -77.0915);            # Bethesda, MD
    routing_preference = "TRAFFIC_AWARE_OPTIMAL",
    departure_time     = "2026-04-27T08:00:00-04:00",
)
parse_duration(route.routes[1].duration)   # seconds
route.routes[1].distanceMeters             # meters

Reuse a client

For longer-running scripts, construct a GoogleMapsClient once so that retry / rate-limit state is shared across calls:

c = GoogleMapsClient(
    key                    = ENV["GOOGLE_MAPS_API_KEY"],
    timeout                = 30,
    retry_timeout          = 60,
    queries_per_second     = 60,
    queries_per_minute     = 6000,
    retry_over_query_limit = true,
    experience_id          = "my-analytics-id",
)

geocode("1600 Amphitheatre Pkwy, Mountain View, CA"; client = c)
elevation((27.9881, 86.9250); client = c)

Per-call overrides (key=, timeout=) do not mutate the shared client; they share the rate-limit state.

Enterprise signing uses client_id + client_secret:

c = GoogleMapsClient(
    client_id     = "gme-myorg",
    client_secret = "vNIXE0xscrmjlyV-12Nj_BvUPaw=",
)

Exceptions

  • ApiError(status, message) — non-OK status in the response envelope (e.g. "OVER_QUERY_LIMIT", "REQUEST_DENIED").
  • HTTPError(status) — unexpected HTTP status on endpoints without an envelope.
  • TransportError(cause) — network / DNS / unexpected exception.
  • RequestTimeout() — total elapsed time exceeded client.retry_timeout.

Where to next

  • Plotting — mapplot, and using any response directly with GeoInterface or GeoMakie.
  • Types — features(response) turns any response into a normalised iterable of GMFeatures.
